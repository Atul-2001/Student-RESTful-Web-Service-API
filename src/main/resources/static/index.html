Student RESTful Web Service API



Base URI

  • /api


Fake Data URI

  • GET  - /api/fake/data   ->  Adds dummy data


Student

Endpoints

  • POST    -  /api/student       ->   Add a new Student
  • PUT     -  /api/student/{id}  ->   Update given student
  • GET     -  /api/students      ->   Get all students
  • GET     -  /api/student/{id}  ->   Get specific student
  • DELETE  -  /api/students      ->   Delete all students
  • DELETE  -  /api/student/{id}  ->   Delete given student

Input Format

                            {
                                "id"     : "[student-id (numerical value only)]", // Optional in POST, but compulsory in PUT
                                "name"   : "[student-name]",
                                "dob"    : "[student-date-of-birth]",
                                "email"  : "[student-email]",
                                "mobile" : "[student-mobile-no]",
                                "course" : {
                                                "id"       : "[course-id][eg:- C101]",
                                                "name"     : "[course-name]",
                                                "duration" : "[course-duration][eg:- 3 years]",
                                                "fees"     : "[course-fees][eg:- 1000/- per year]"
                                            }
                            }


Course

  • PUT     -  /api/course/{id}  ->   Update given course
  • GET     -  /api/courses      ->   Get all courses
  • GET     -  /api/course/{id}  ->   Get specific course
  • DELETE  -  /api/courses      ->   Delete all courses
  • DELETE  -  /api/course/{id}  ->   Delete given course

Input Format

                            {
                                "id"       : "[course-id][e.g:- C101]", // Optional in POST, but compulsory in PUT
                                "name"     : "[course-name]",
                                "duration" : "[course-duration][eg:- 3 years]",
                                "fees"     : "[course-fees][eg:- 1000/- per year]"
                            }
